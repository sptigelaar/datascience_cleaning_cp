Tidy data code book
=======================

The tidy data set contains the following data:

- Activity - string describing the activity of the subject
	WALKING
	WALKING_UPSTAIRS
	WALKING_DOWNSTAIRS
	SITTING
	STANDING
	LAYING

- Subject_number - integer describing the identification number of the subject
	1:30

 The following measures are reported as averages (mean) per subject-activity combination.
